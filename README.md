# 📊 Sales Data Analysis Project

## 📌 Overview
This project focuses on analyzing sales data using Python and Power BI to extract meaningful business insights. It includes data cleaning, exploratory data analysis (EDA), and interactive dashboard creation.

---

## 🛠️ Tools & Technologies Used
- Python (Pandas, Matplotlib, Seaborn)
- Microsoft Power BI
- Microsoft Excel
- Jupyter Notebook

---

## 📁 Project Files
- 📓 Python Notebook (.ipynb) – Data Cleaning & EDA
- 📊 Power BI Dashboard (.pbix) – Interactive Visualization
- 📄 PowerPoint Presentation (.pptx) – Project Summary

---

## 🔍 Key Steps Performed
- Data Cleaning and preprocessing
- Handling missing values and duplicates
- Exploratory Data Analysis (EDA)
- Data visualization using Python
- Interactive dashboard creation in Power BI

---

## 📈 Key Insights
- Sales performance varies across different regions
- Certain customer segments contribute more to revenue
- Year-wise sales trends help in business forecasting
- Dashboard helps in quick decision-making

---

## 🎯 Outcome
This project helped in improving skills in data analysis, visualization, and business intelligence using real-world datasets.
